# NUShelter

## Description 
NUShelter is a campus wide umbrella sharing initiative that lets NUS students donate, borrow and return umbrellas easily through a live campus map and identity-linked system.

## Features

1. Umbrella Donation, Borrowing and Returning
2. Live Availability Map
3. Overdue Notifications and Suspension
4. User Profile Summary Centre
5. Umbrella Impact Card
6. Thank You Wall
